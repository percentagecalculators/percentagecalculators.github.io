How to Add a Percentage to a Number

Adding a percentage to a number is essential for calculating raises, markups, and growth. This tool from Percentage Calculators performs the calculation instantly, showing both the final value and the increase amount.

Percentage Increase Formula

New Value = Original Value × (1 + Percentage ÷ 100)

Example: To increase 150 by 20%, calculate 150 × 1.20 = 180. The increase amount is 30.

Common Uses

  • Calculating salary after a raise
  • Adding markup to wholesale prices
  • Computing investment returns
  • Adjusting prices for inflation

If you need to subtract a percentage instead of adding, use our percentage decrease calculator which works the same way but in reverse – perfect for calculating discounts, depreciation, or any value reduction.
