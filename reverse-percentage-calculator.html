How to Find the Original Number from a Percentage

Sometimes you know a partial value and its percentage but need the original whole. This tool from Percentage Calculators works backwards to find it.

Reverse Percentage Formula

Original Number = (Known Value ÷ Percentage) × 100

Example

If $150,000 profit represents 25% of revenue, what is total revenue? ($150,000 ÷ 25) × 100 = $600,000.

Common Uses

  • Find original price before discount
  • Calculate pre-tax amounts from tax paid
  • Find total from sample percentage
  • Determine base salary from bonus percentage
