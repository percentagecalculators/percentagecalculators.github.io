Calcola la media di più percentuali

Fare la media di più percentuali è un compito comune. Questo calcolatore gestisce un numero illimitato di valori percentuali.

Formula della media semplice

Percentuale Media = (Somma di tutte le percentuali) ÷ (Numero di percentuali)

Esempio di calcolo

Percentuali voti: 85%, 92%, 78%, 90%, 80%

  1. Somma: 85 + 92 + 78 + 90 + 80 = 425
  2. Conteggio: 5 valori
  3. Media: 425 ÷ 5 = 85%

Quando usare la media semplice

Scenari con peso uguale

La media semplice funziona bene quando tutti i valori hanno la stessa importanza:

  • Test con peso uguale – Cinque test dove ognuno conta uguale
  • Valutazioni performance – Valutazioni mensili con peso uguale
  • Risposte sondaggio – La percentuale di ogni rispondente conta uguale

Media semplice vs Media ponderata

Aspetto Media Semplice Media Ponderata
Formula Somma ÷ Conteggio Somma(valore × peso) ÷ Somma(pesi)
Assunzione Tutti i valori ugualmente importanti I valori hanno importanze diverse
Esempio 5 test uguali Esame parziale 30%, finale 70%
